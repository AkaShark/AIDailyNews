### [🎮 Delta：iOS设备上的多合一经典视频游戏模拟器](https://github.com/rileytestut/Delta)

来源：Trending Swift repositories on GitHub today · GitHub

发布时间：2024-04-22 09:00:22

Delta是一款适用于非越狱iOS设备的多合一经典视频游戏模拟器，它支持多种经典视频游戏系统，包括Game Boy Advance、Nintendo 64和Nintendo DS。Delta具有准确、全速的模拟功能，漂亮、原生的用户界面，支持控制器，自定义控制器皮肤，保存状态，快速转发，Delta同步，按住按钮，3D/触觉触控，作弊代码，陀螺仪支持和麦克风支持。

### [💻 Code Llama：用于代码的大型语言模型](https://github.com/meta-llama/codellama)

来源：Trending repositories on GitHub today · GitHub

发布时间：2024-04-22 06:13:25

Code Llama 是一个大型语言模型系列，专门用于代码。它提供基础模型、Python 特化模型和指令遵循模型，参数分别为 7B、13B 和 34B。这些模型支持高达 100k 令牌的输入上下文，并且可以执行代码填充和安全命令执行。
