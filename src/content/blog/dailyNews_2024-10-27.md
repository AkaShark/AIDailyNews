## Apple News

### [💰 苹果悬赏 100 万美元，寻找私人 AI 云漏洞](https://techcrunch.com/2024/10/24/apple-will-pay-security-researchers-up-to-1-million-to-hack-its-private-ai-cloud/)

来源：Hacker News - Newest: "apple"

发布时间：2024-10-26 13:51:10

苹果公司宣布，将向发现其私人 AI 云漏洞的安全研究人员支付高达 100 万美元的赏金。该漏洞赏金计划旨在提高私人 AI 云的安全性和隐私性，并鼓励研究人员报告潜在的漏洞。苹果表示，将根据漏洞对用户数据和推理请求数据的影响程度，向研究人员支付不同金额的赏金。

## Tech News

### [🌟 衡量开发者生产力的新指标：Diff Authoring Time](https://engineering.fb.com/2024/10/25/developer-tools/diff-authoring-time-dat-measuring-developer-productivity-meta/)

来源：Engineering at Meta

发布时间：2024-10-26 00:32:59

Meta 提出了一种新的指标 Diff Authoring Time (DAT) 来衡量开发人员的生产力。DAT 衡量的是从开始编写代码到提交代码更改所花费的时间。Meta 认为，DAT 可以更好地反映开发人员的实际工作效率，因为它考虑了代码审查和合并等因素。

### [🚀 GitHub Copilot 提升了 Visual Studio 中的 C++ 代码补全功能](https://github.blog/changelog/2024-10-25-improving-github-copilot-c-completions-in-visual-studio)

来源：GitHub Changelog

发布时间：2024-10-26 04:02:30

GitHub Copilot 现已提升了 Visual Studio 中的 C++ 代码补全功能，改进了补全建议的准确性和相关性，并加入了对更多 C++ 特性的支持，从而帮助开发者更轻松、高效地编写 C++ 代码。
