来源：Latest News - Apple Developer

发布时间：2024-03-01 05:40:13

### [⚡️ 菲斯克暂停新电动汽车车型研发，寻求更多资金](https://www.engadget.com/fisker-halts-work-on-new-ev-models-until-it-finds-more-money-140050091.html?src=rss)

来源：Engadget is a web magazine with obsessive daily coverage of everything new in gadgets and consumer electronics

发布时间：2024-03-01 19:58:09

电动汽车制造商 Fisker 公布了其未来计划以及 2023 年和第四季度的初步收益，情况不容乐观。该公司计划裁员 15%，即近 200 人，因为它从直接面向消费者的模式转向经销商合作伙伴模式。该公司将暂停对即将推出的车型的所有投资，只有在与另一家汽车制造商合作的情况下才会恢复。

该公司的第四季度收入从第三季度的 1.283 亿美元增加到 2.001 亿美元。然而，其毛利率为负 35%，每股亏损 1.23 美元。其在市场上的唯一一款电动汽车 Ocean SUV 也生产了 10,193 辆，但交付了 4,929 辆。

这家汽车制造商最初在 1 月份推出了其向经销商合作伙伴模式的转型，并声称已收到来自北美和欧洲 250 家经销商的兴趣，并签署了 13 项协议。菲斯克董事长兼首席执行官亨里克·菲斯克在一份声明中表示：“我们意识到该行业已进入一个动荡且不可预测的时期”。“有了这种理解并吸取了 2023 年的教训，我们制定了一项计划，在为又一个困难的年份做好准备时精简公司。我们已调整了对 2024 年的前景，使其比 2023 年更加保守。”该公司计划在全球范围内交付 20,000 至 22,000 辆 Ocean 车型。

Fisker 目前正在与“一家大型汽车制造商”就未来电动汽车的投资和联合生产进行谈判。这意味着先前宣布的汽车生产，例如带有超大杯架和指定牛仔帽空间的 Alaska EV 皮卡，将无限期搁置。Fisker 原计划于 2025 年初开始生产 Alaska EV 皮卡。

### [🌟 BitNet：大语言模型的1位Transformer](https://github.com/kyegomez/BitNet)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2024-03-02 07:45:33

BitNet是一种新的Transformer架构，它使用1位线性层来显着减少模型大小和计算成本。与使用32位浮点精度的标准Transformer相比，BitNet在保持类似性能的同时，将模型大小减少了90%以上，推理时间减少了80%以上。BitNet的实现非常简单，只需要用BitLinear模块替换Transformer中的线性投影。BitNet在各种NLP任务上都取得了有竞争力的结果，包括文本分类、问答和语言生成。

### [📸 VisionCamera：React Native的高性能相机库](https://github.com/mrousavy/react-native-vision-camera)

### [🌟 Pingora：构建快速、可靠且可编程的网络系统](https://github.com/cloudflare/pingora)

来源：Trending Rust repositories on GitHub today · GitHub

发布时间：2024-03-02 08:53:35

Pingora 是一个 Rust 框架，用于构建快速、可靠且可编程的网络系统。Pingora 经过实战检验，多年来每秒处理超过 4000 万个互联网请求。Pingora 的特点包括：异步 Rust、HTTP 1/2 端到端代理、TLS over OpenSSL 或 BoringSSL、gRPC 和 websocket 代理、平滑重新加载、可定制的负载均衡和故障转移策略以及对各种可观测性工具的支持。使用 Pingora 的原因包括：安全性是您的首要任务、您的服务对性能敏感、您的服务需要广泛的定制。
