### [📝 Userscripts Safari: Safari 的开源用户脚本编辑器](https://github.com/quoid/userscripts)

来源：Trending Swift repositories on GitHub today · GitHub

发布时间：2024-08-20 05:33:53

Userscripts Safari 是一款适用于 iOS 和 macOS 的开源用户脚本编辑器。它允许用户创建、编辑和管理 Safari 中的用户脚本。该应用程序具有直观的界面，支持多种用户脚本元数据，并提供 API 方法来与脚本进行交互。它还支持脚本更新、代码提示和语法高亮。
